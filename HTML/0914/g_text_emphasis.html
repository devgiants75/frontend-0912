중요(Emphasis)

em태그(emphasis)

: 텍스트를 강조할 때 사용하는 태그

i태그(italic)

: 텍스트를 기울여서 표현할 때 사용하는 태그

em태그와 i태그의 차이

- em태그: 문장에서 흐름상 특정 부분을 강조
- i태그: 단순히 다른 텍스트와 구별하기 위해 기울여서 표시

강조(Strong importance)

strong태그

: 텍스트를 굵게 강조할 때 사용하는 태그

b태그(bold)

: 텍스트를 굵게 표시할 때 사용하는 태그

strong태그와 b태그의 차이

- strong태그: 경고 OR 주의사항처럼 중요한 내용을 강조
- b태그: 단순히 글자만 굵게 표시할 때
