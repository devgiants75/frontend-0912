HTML 속성 : 요소에 대한 추가 정보를 제공 - 모든 HTML요소는 속성을 가질 수 있다. - 항상 시작 태그에 지정. - 일반적으로 name='value'와 같은 이름='값' 쌍으로 구성 !HTML 속성 사용 시 주의사항! 1. 항상 소문자 속성을 사용 2. 요소 이름 다음에 바로 나오는 속성은 요소 이름과 속성 사이에 공백이 필요 하나 이상의 속성들이 있는 경우에는 속성들 사이에 공백이 있어야 함. 3. 속성 이름 다음에는 등호(=)가 붙는다. 4. 속성 값은 열고 닫는 따옴표로 감싸야 함. HTML 속성의 종류 1. href 속성(hyper reference: 하이퍼 참조) 네이버 스포츠 야구 2. src 속성(source: 소스) 3. alt 속성(alter: 바꾸다) # 이미지 경로 지정 # 1. 절대 경로 : 웹 사이트의 루트 디렉토리에서 시작하여 전체 파일 경로를 지정하는 방식 - 예. https://cdn.pixabay.com/photo/2016/02/18/18/37/puppy-1207816_1280.jpg - 웹 페이지의 위치와 관계없이 항상 같은 리소스를 참조 2. 상대 경로 : 현재 문서의 위치를 기준으로한 파일의 경로를 지정하는 방식 - 상대경로는 파일의 위치에 따라 지정하는 방식이 다르다. 1) 같은 디렉토리 파일: same-directory.jpg 2) 하위 디렉토리 파일: images/images-Folder.jpg 3) 상위 디렉토리 파일: ../diff-directory.jpg 4. title속성 : 링크에 대한 추가 정보를 나타냄. 네이버 바로가기
