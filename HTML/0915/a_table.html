HTML Table

  • 1. 테이블(표) : 행(row), 열(column) 그리고 셀(cell)로 이루어짐.
  • 행(row): 가로 / 열(column): 세로
  • 셀(cell): 표의 내용이 들어가는 한 칸을 의미

table태그, caption태그

표를 만드는 태그, 표를 생성 표의 제목 또는 설명을 삽입 - 제목이 표의 위쪽 중앙에 표시 - 선택적 항목, 주로 스크린 리더 사용자를 위해 작성

tr, td, th 태그

tr태그: table row(각 행) td태그: table data(일반적인 테이블 셀) th태그: table header(테이블의 제목 셀, 기본적으로 굵은 글씨) # table태그 작성만으로 표 생성 X - 태그 안의 행과 각 행의 셀의 개수를 지정
첫 테이블 생성
1행 1열 1행 2열
2행 1열 2행 2열

표의 구조를 지정하는 태그

  • thead(제목), tbody(본문), tfoot(요약) 태그
  • 표의 구조를 지정할 경우 시각 장애인도 화면 낭독기를 통해 표를 쉽게 이해 가능
  • CSS 사용하여 각각 다른 스타일 적용 가능
  • 테이블 구조 사용 전후에 눈에 보여지는 테이블은 동일하지만 화면 낭독기나 컴퓨터 프로그램이 인지하는 구조가 달라진다.

테이블 병합

rowspan, colspan 속성을 사용하여 행이나 열을 병합 rowspan: 행 병합 colspan: 열 병합

: 특정 셀이 여러 행에 걸쳐 확장되어야 할 때 사용
: rowspan = '2' 적용 시 해당 셀이 아래쪽으로 2행만큼 확장
