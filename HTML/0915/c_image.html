이미지 삽입 태그: img태그

웹 문서에서 이미지를 삽입할 때 가장 기본이되는 태그
- 이미지는 기술적으로 웹 페이지에 삽입 X, 웹 페이지에 링크 O
기본형:

src(source) 속성

  • 이미지 파일의 경로를 지정: 웹 브라우저에 알려주는 역할
  • 동일한 경로: 이미지 파일의 이름만 표기(확장자 포함)
  • 하위 폴더: src속성에 폴더이름을 포함하여 표기
  • 다른 서버 / 웹 사이트의 이미지: Full URL모두 지정

alt(alter) 속성

  • 화면 낭독기 등에서 이미지를 대신하여 읽어 줄 텍스트를 입력
  • 이미지를 제대로 표시할 수 없는 경우에 이미지 대신 텍스트가 나타남.

너비, 높이를 지정하는 속성

width속성: 이미지의 너비를 지정

height속성: 이미지의 높이를 지정

width, height 중 1개만 지정해도 나머지 속성은 비율을 자동으로 계산

너비와 높이를 지정하는 단위

- 퍼센트(%) : 이미지 크기의 값을 퍼센트로 지정 : 현재 웹 브라우저 창의 너비와 높이를 기준으로 이미지의 크기를 결정 : 웹 브라우저 창의 너비와 높이에 따라 상대적으로 크기가 결정

- 픽셀(px) : 디지털 이미지의 기본단위 : 한 이미지는 수많은 작은 점들로 구성되어 있으며, 이 작은 점을 픽셀이라고 부름. : 절대적인 단위 : px 단위 생략 가능
