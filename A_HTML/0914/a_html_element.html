블록 레벨 요소 vs 인라인 레벨 요소

블록 레벨 요소

: 웹 페이지 상에서 블록을 만드는 요소

- 앞뒤 요소 사이에 새로운 줄(Line)을 만들고 나타냄. - 웹 페이지의 구조를 형성하는 요소 - 콘텐츠 전 후로 줄 바꿈이 있음. - 가로 폭 전체를 사용

특징 - 인라인 요소에 중첩 불가능 - 다른 블록 레벨 요소에는 중첩 가능

블록 레벨 요소 예시

    1. 인라인 레벨 요소

      : 문서의 한 단락같은 큰 범위에서는 적용 될 수 없고 문장, 단어 같은 작은 부분에서만 적용 가능

      - 인라인 요소는 새로운 줄(Line)을 만들지 않는다. - 웹 페이지의 텍스트 내에서 사용되는 요소 - 줄 바꿈 없이 텍스트의 흐름에 따라 나타남. - 필요한 만큼의 너비만 사용

      인라인 레벨 요소 예시 // 예외적으로 너비와 높이 속성을 가질 수 있음
