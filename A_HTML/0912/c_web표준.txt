# 웹 표준 #

# 웹 표준 정의
: 웹에서 사용되는 표준화된 기술이나 규칙을 의미

# 웹 표준 중요성
: 다양한 기기와 브라우저에서 일관된 웹 경험을 제공
: 웹 접근성 보장
: 유지 보수 용이

# 웹 접근성
: 모든 사용자가 웹 사이트와 도구에 동등하게 접근할 수 있게 만드는 것
: 장애인, 고령자를 포함한 모든 사용자의 웹 이용 확장

# 웹 표준의 주요 기술
# HTML: 웹 페이지의 기본 구조와 내용을 정의 - HTML5
# CSS: 웹 페이지의 디자인 - CSS3
# JavaScript: 웹 페이지에 동적인 요소를 추가하는 데 사용
# DOM
: 웹 페이지의 객체와 구조를 표현하고
  , JS가 페이지와 상호작용 하도록 도와주는 인터페이스
# Accessibility Guidelines
: 웹 콘텐츠를 모든 사용자들이 접근하고 이해할 수 있도록 하는 웹 접근성 가이드라인

# interface(인터페이스)
# : 상호작용(을 할 수 있도록 미리 규정한 내용)
