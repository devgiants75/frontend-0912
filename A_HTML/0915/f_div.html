div태그

HTML에서 가장 흔하게 사용되는 구조적 요소 중 하나
: 웹 페이지의 여러 부분을 나누거나 묶는 데 사용
: 주로 CSS(스타일), JavaScript(기능)을 적용하기 위해 사용

class, id 속성

class, id속성을 사용하여 HTML 태그에 스타일링이나 동작을 지정 가능

class 속성
  • 하나의 HTML요소에 여러 클래스를 할당 가능 (띄어쓰기로 구분)
  • 여러 요소에 동일한 클래스를 할당 가능
  • 클래스 구문을 사용하려면 마침표(.) 문자 뒤에 클래스 이름을 입력 - 그 다음 중괄호 {} 내에서 CSS 속성을 정의

id 속성
  • 각 HTML 요소는 고유한 id를 가져야한다.
  • id 구문을 사용하려면 해시문자(#) 문자 뒤에 id 이름을 입력 - 그 다음 중괄호 {} 내에서 CSS속성을 정의
